# ROAMR — MVP (web app)

Mobile-first React app built from the ROAMR HiFi Figma screens.
React + Vite + Tailwind + React Router.

## Run locally
```bash
npm install
npm run dev
```
Open the printed localhost URL. Best viewed at phone width (DevTools device toolbar).

## Build / deploy
```bash
npm run build      # outputs to dist/
```
Deploy `dist/` to Netlify (drag-and-drop or `netlify deploy --prod`).
