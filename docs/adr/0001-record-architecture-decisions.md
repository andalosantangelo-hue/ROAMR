# 1. Record architecture decisions

Date: 2026-06-18 · Status: Accepted

## Context
ROAMR is a fast-moving, mostly solo/duo build. Decisions were scattered across spec docs.

## Decision
Keep short Architecture Decision Records (ADRs) in `docs/adr/`, one file per decision,
numbered sequentially. Each records context, the decision, and consequences.

## Consequences
New contributors can see *why* the system is the way it is without spelunking chat history.
